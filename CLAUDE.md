# 이후 (소설가) 네이버 SEO 최적화 도구

## 프로젝트 목적
네이버 검색에서 "이후" 키워드 검색 시 소설가 이후의 정보가 상위 노출되도록 돕는 SEO 분석·모니터링 도구

## 대상 정보
- 이름: 이후 (소설가)
- 네이버 프로필: https://search.naver.com/search.naver?...&os=215161&query=%EC%9D%B4%ED%9B%84
- 위키백과: https://ko.wikipedia.org/wiki/%EC%9D%B4%ED%9B%84_(%EC%86%8C%EC%84%A4%EA%B0%80)
- 홈페이지: https://xn--hu5b23z.com/

## 주요 기능
1. 네이버 검색 순위 모니터링 (현재 순위 체크)
2. 페이지 SEO 분석 (홈페이지, 위키 등)
3. 개선 권고안 생성
4. 순위 변동 히스토리 기록

## 기술 스택
- Python 3
- requests, BeautifulSoup4, rich (CLI 대시보드)
- SQLite (순위 히스토리 저장)
